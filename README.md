# SimpleMusicPlayer
## 概要
自分用のシンプルなデザインのミュージックプレイヤー

[openCV](https://github.com/opencv/opencv), [cvui](https://github.com/Dovyski/cvui), [DxLib](http://dxlib.o.oo7.jp/), [Boost](https://www.boost.org/)を使用して製作しました。
